A simple Alfred Workflow to search your GoodLinks.app links using the local REST API.

## Usage
Type `link` and continue typing to search.

## Setup
This workflow uses the GoodLinks local REST API.
1. Open GoodLinks on your Mac.
2. Go to **Settings > Advanced**.
3. Enable **Local API**.

The workflow will connect to the default port `9428`.

Download it [here](https://github.com/kbro237/goodlinks-alfred/releases/latest).
